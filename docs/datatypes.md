| ----------- | ---------- |
| smallint    | 2 bytes    |
| integer     | 4 bytes    |
| bigint      | 8 bytes    |
| decimal     | variable   |
| numeric     | variable   |
| smallserial | 2 bytes    |
| serial      | 4 bytes    |
